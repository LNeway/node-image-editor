## 问题分析

### 问题：颜色设置后预览不刷新

**现象：**
- 首次设置颜色（红色）可以正常渲染
- 后续修改其他颜色时，预览图像仍显示第一次设置的颜色

**测试流程：**
1. ✅ 设置红色 → 渲染正确 (255,0,0)
2. ❌ 设置绿色 → 仍显示红色 (255,0,0)
3. ❌ 设置蓝色 → 仍显示红色 (255,0,0)
4. ❌ 设置白色 → 仍显示红色 (255,0,0)
5. ❌ 设置黑色 → 仍显示红色 (255,0,0)
6. ❌ 设置黄色 → 仍显示红色 (255,0,0)

**根本原因：**
- 颜色参数变更后未触发节点重新执行
